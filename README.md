# Desafios de Código em C 🚀

Este repositório foi criado para armazenar meus exercícios, desafios de lógica e estudos na linguagem C.

## 📌 Desafios Concluídos:

### 01. Inversão de Frase com Espaços Adaptados
* **Conceito treinado:** Loops `for`, manipulação de strings (`char[]`), controle de índices (`i` e `j`) e funções da `string.h`.
* **O Desafio:** Inverter a frase *"a base do teto desaba"* mantendo os espaços exatamente nas mesmas posições da string original, gerando o resultado *"abased otet od esab a"* (adaptando-a pra ficar idêntica a frase antes de inverter).
